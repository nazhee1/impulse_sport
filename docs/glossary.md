# Краткий словарь терминов

## Интернет-магазин

Веб-система для продажи товаров через интернет.

## Информационная система

Система, предназначенная для хранения, обработки и передачи информации.

## Frontend

Пользовательский интерфейс, с которым работает посетитель сайта.

## Backend

Серверная часть приложения, которая обрабатывает запросы и работает с базой данных.

## Клиент-серверная архитектура

Подход, при котором интерфейс и серверная логика разделены на разные части.

## API

Интерфейс взаимодействия между программными компонентами.

## REST API

Подход к построению веб-интерфейсов обмена данными через HTTP-запросы.

## JSON

Текстовый формат представления данных, часто используемый в веб-разработке.

## База данных

Организованное хранилище информации.

## Реляционная база данных

База данных, в которой информация хранится в виде таблиц и связей между ними.

## Таблица

Структура хранения данных в базе данных.

## Запись

Одна строка таблицы с конкретным набором данных.

## Первичный ключ

Поле, которое однозначно идентифицирует запись в таблице.

## Внешний ключ

Поле, которое связывает одну таблицу с другой.

## Нормализация

Способ организации данных для уменьшения дублирования.

## Каталог

Раздел магазина, в котором представлены товары.

## Фильтрация

Отбор товаров по заданным условиям.

## Поиск

Нахождение товаров по ключевому слову или запросу.

## Карточка товара

Страница с подробной информацией о конкретном товаре.

## Корзина

Раздел, в котором временно хранятся выбранные пользователем товары.

## Заказ

Оформленная пользователем покупка.

## Администратор

Пользователь с расширенными правами управления системой.

## Валидация

Проверка корректности введенных данных.

## Аутентификация

Проверка личности пользователя при входе в систему.

## Авторизация

Определение прав доступа пользователя после входа в систему.

## Адаптивный интерфейс

Интерфейс, корректно работающий на разных размерах экрана.

## Прототип

Учебная или предварительная версия системы, демонстрирующая основные функции.
