## Вариант выступления

Здравствуйте, уважаемые члены комиссии.

Меня зовут Мисриев Нажмудин Альбертович, я студент группы 05-23.ИСИП.ОФ.11.

Тема моей дипломной работы: «Разработка интернет-магазина спортивных товаров».

Актуальность темы обусловлена тем, что рынок электронной коммерции продолжает активно развиваться, а покупатели все чаще выбирают спортивные товары через интернет. При этом для эффективной работы интернет-магазина недостаточно просто разместить товары на сайте. Необходимо обеспечить удобный каталог, поиск и фильтрацию, хранение данных о пользователях и заказах, а также инструменты администрирования.

Целью работы является разработка информационной системы интернет-магазина спортивных товаров, которая позволяет автоматизировать основные процессы онлайн-продаж.

Для достижения этой цели были поставлены следующие задачи:

1. проанализировать предметную область;
2. определить требования к системе;
3. спроектировать архитектуру приложения;
4. разработать структуру базы данных;
5. спроектировать пользовательский интерфейс;
6. описать API для взаимодействия клиентской и серверной частей;
7. реализовать демонстрационный прототип системы.

Объектом исследования является процесс онлайн-продажи спортивных товаров.

Предметом исследования являются методы и средства разработки веб-приложений для автоматизации продаж.

В рамках проекта была выбрана клиент-серверная архитектура. Пользователь взаимодействует с веб-интерфейсом, клиентская часть отправляет запросы к серверу через REST API, а сервер обрабатывает данные и работает с базой данных PostgreSQL.

В системе предусмотрены три основные роли: гость, зарегистрированный пользователь и администратор.

Гость может просматривать каталог и искать товары.
Зарегистрированный пользователь может добавлять товары в корзину и оформлять заказ.
Администратор может управлять каталогом, изменять информацию о товарах и контролировать статусы заказов.

Основной функционал системы включает:

1. просмотр каталога спортивных товаров;
2. поиск и фильтрацию по категориям и цене;
3. просмотр карточки товара;
4. добавление товара в корзину;
5. оформление заказа;
6. хранение данных о заказах;
7. администрирование каталога и заказов.

При проектировании базы данных были выделены основные сущности: пользователи, категории, бренды, товары, изображения товаров, корзины, заказы, позиции заказа, адреса, платежи и отзывы. Такая структура позволяет избежать дублирования данных и поддерживать целостность информации.

В пользовательской части был создан прототип интерфейса интернет-магазина. В нем реализованы главная страница, каталог товаров, фильтрация, поиск, добавление в корзину и расчет итоговой суммы заказа. Также предусмотрен блок административной аналитики, который демонстрирует возможность дальнейшего расширения системы.

Практическая значимость проекта заключается в том, что разработанное решение может использоваться как основа для интернет-магазина малого или среднего бизнеса в сфере спортивных товаров. Система помогает структурировать каталог, упростить взаимодействие с клиентом и автоматизировать обработку заказов.

В качестве перспектив развития проекта можно выделить:

1. реализацию полноценного backend;
2. подключение реальной базы данных;
3. авторизацию пользователей;
4. интеграцию онлайн-оплаты;
5. создание административной панели;
6. добавление системы отзывов и рекомендаций.

Таким образом, цель дипломной работы была достигнута, поставленные задачи решены, а разработанный проект демонстрирует применение современных веб-технологий для автоматизации интернет-магазина спортивных товаров.
