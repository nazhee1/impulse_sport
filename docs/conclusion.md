# Заключение

В рамках дипломного проекта была рассмотрена задача разработки интернет-магазина спортивных товаров. В ходе выполнения работы была проанализирована предметная область, определены особенности онлайн-продажи спортивных товаров и сформулированы требования к будущей системе.

На этапе проектирования была выбрана клиент-серверная архитектура, определены роли пользователей, спроектирована структура базы данных и подготовлен пример REST API для взаимодействия между компонентами системы. Также был продуман пользовательский интерфейс, обеспечивающий удобный просмотр каталога, поиск товаров, фильтрацию и работу с корзиной.

Практическая часть проекта представлена демонстрационным прототипом интернет-магазина `SportStore`, который показывает основные пользовательские сценарии: просмотр каталога, поиск, фильтрацию, добавление товаров в корзину и расчет итоговой суммы заказа. Проведенное тестирование подтвердило корректность работы базовой логики прототипа.

Таким образом, цель работы достигнута: разработана концепция и прототип информационной системы интернет-магазина спортивных товаров, охватывающей ключевые процессы онлайн-продаж. Полученные результаты подтверждают, что выбранный подход является целесообразным и может быть использован как основа для дальнейшей реализации полноценного веб-приложения.

Перспективами развития проекта являются реализация серверной части, подключение базы данных, организация авторизации пользователей, интеграция онлайн-оплаты, развитие административного модуля и расширение аналитических возможностей системы.
