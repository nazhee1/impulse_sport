# Аннотация проекта

Дипломный проект посвящен разработке интернет-магазина спортивных товаров `SportStore`. Основной целью работы является создание информационной системы, обеспечивающей просмотр каталога товаров, поиск и фильтрацию, добавление товаров в корзину, оформление заказов и административное сопровождение процесса продаж.

В рамках проекта были выполнены анализ предметной области, формирование требований, проектирование архитектуры приложения, разработка модели базы данных, описание REST API и создание пользовательского интерфейса. Практическая часть представлена запускаемой учебной веб-системой, в которой frontend взаимодействует с backend API на Node.js. Для хранения данных в демонстрационной версии используются JSON-файлы.

Разработанная система демонстрирует основные бизнес-процессы интернет-магазина спортивных товаров: работу каталога, поиск, фильтрацию, корзину, оформление заказа, историю заказов и административное изменение статусов. Проект может быть использован как основа для дальнейшего развития полноценного веб-приложения с PostgreSQL, авторизацией и полноценной административной панелью.
