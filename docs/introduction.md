# Введение

Выпускная квалификационная работа посвящена разработке интернет-магазина спортивных товаров. Выбор данной темы обусловлен тем, что электронная коммерция продолжает активно развиваться, а интернет-магазины становятся одним из основных каналов продаж для компаний различного масштаба. Особенно это актуально для сферы спортивных товаров, где покупателю важно быстро находить нужную категорию продукции, сравнивать характеристики, оценивать стоимость и оформлять заказ в удобной форме.

Современный пользователь ожидает от интернет-магазина не только наличия каталога, но и удобной системы поиска, фильтрации, отображения характеристик товара, корзины, оформления заказа и получения информации о статусе покупки. Для владельца магазина, в свою очередь, важны централизованное хранение данных, управление ассортиментом, контроль остатков, обработка заказов и возможность дальнейшего масштабирования системы.

Актуальность темы заключается в необходимости автоматизации процессов продажи спортивных товаров через веб-интерфейс. Разработка подобной системы позволяет объединить в одном решении задачи публикации каталога, взаимодействия с клиентом, хранения информации о товарах и заказах, а также администрирования основных бизнес-процессов. С учебной точки зрения проект является показательным, так как охватывает несколько ключевых направлений веб-разработки: проектирование интерфейсов, разработку клиентской и серверной архитектуры, работу с базами данных и моделирование прикладной логики.

Целью дипломного проекта является разработка информационной системы интернет-магазина спортивных товаров, предназначенной для автоматизации онлайн-продаж, упрощения выбора товаров и организации процесса оформления заказа.

Для достижения поставленной цели были сформулированы следующие задачи:

1. Изучить предметную область и особенности онлайн-продажи спортивных товаров.
2. Проанализировать требования к системе со стороны пользователя и администратора.
3. Определить состав функциональных и нефункциональных требований.
4. Спроектировать архитектуру интернет-магазина.
5. Разработать логическую структуру базы данных.
6. Описать взаимодействие компонентов системы через API.
7. Создать демонстрационный прототип пользовательского интерфейса.
8. Подготовить материалы для тестирования и защиты проекта.

Объектом исследования является процесс онлайн-продажи спортивных товаров.

Предметом исследования являются методы и средства проектирования и разработки веб-приложений, применяемые для создания интернет-магазина.

Практическая значимость проекта состоит в том, что разработанное решение может использоваться как основа для реального интернет-магазина малого или среднего бизнеса. Система позволяет структурировать каталог товаров, организовать хранение данных о пользователях и заказах, упростить процесс покупки и заложить основу для дальнейшего развития сервиса.

В ходе выполнения работы использовались методы анализа предметной области, структурного проектирования, моделирования данных, прототипирования пользовательского интерфейса и функционального тестирования. В качестве концептуального технологического стека рассматриваются HTML, CSS и JavaScript для клиентской части, серверная часть на Node.js или Python, а также реляционная база данных PostgreSQL.

Структура работы включает введение, три главы, заключение и приложения. В первой главе рассматривается предметная область и анализируются особенности интернет-магазина спортивных товаров. Во второй главе приводится проектирование архитектуры системы, базы данных и интерфейса. В третьей главе описывается реализация прототипа, тестирование и перспективы развития проекта. В заключении формулируются основные результаты выполненной работы.
