# Руководство пользователя

## Назначение системы

`SportStore` предназначен для работы интернет-магазина спортивных товаров. В системе есть каталог, карточки товаров, корзина, оформление заказов, личный кабинет покупателя, рабочий кабинет менеджера и админ-панель.

## Запуск

1. Откройте терминал в папке проекта `sports-store-diploma`.
2. Выполните команду:

```bash
npm start
```

3. Откройте сайт по адресу сервера на порту `3030`.

## Данные для входа

- Менеджер: `manager` / `manager`
- Администратор: `admin` / `admin`

## Сценарии покупателя

### Просмотр каталога

На главной странице доступен каталог спортивных товаров. Гость может просматривать товары и карточки, но для добавления в корзину и оформления заказа нужно войти в аккаунт.

### Карточка товара

Кнопка `Подробнее` открывает отдельную страницу товара. На ней показаны описание, цена, остаток на складе, рейтинг, отзывы и похожие товары.

### Корзина

После входа покупатель может добавить товар в корзину. Если товара нет в наличии, кнопка добавления отключается. В корзине можно менять количество, удалять позиции и перейти к оформлению заказа.

### Оформление заказа

В форме заказа указываются телефон, адрес доставки, способ оплаты и комментарий. После подтверждения заказ сохраняется, а остатки товаров уменьшаются.

### Личный кабинет

В кабинете покупателя доступны личные данные, история заказов, избранные товары и недавно просмотренные карточки.

## Сценарии менеджера

Менеджер видит заказы магазина, может отслеживать их статусы и контролировать товары с низкими остатками.

## Сценарии администратора

Администратор управляет товарами, категориями, заказами и сотрудниками. В админ-панели можно добавлять и редактировать товары, загружать изображение товара, менять остатки, создавать категории и удалять ненужные записи.

## Проверка перед показом

Перед демонстрацией рекомендуется открыть главную страницу, войти под каждой ролью, добавить товар в корзину, оформить заказ и убедиться, что заказ появился в кабинете и админ-панели.
