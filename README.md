# Импульс Спорт

«Импульс Спорт» - учебный интернет-магазин спортивных товаров, разработанный
в рамках дипломного проекта по теме «Разработка интернет-магазина спортивных товаров».

Проект включает каталог товаров, карточки товаров, корзину, оформление заказа,
личные кабинеты для покупателя, менеджера и администратора, а также управление
товарами, категориями, остатками и заказами.

## Структура проекта

- `prototype/` - пользовательский интерфейс сайта.
- `src/` - серверная логика и API.
- `data/` - данные каталога, заказов, пользователей и отзывов.
- `tests/` - автоматическая проверка основных сценариев.
- `docs/` - материалы пояснительной записки и защиты.
- `output/` - подготовленные печатные версии дипломных материалов.
- `server.js` - точка запуска приложения.
- `package.json` - команды проекта.

## Основные возможности

1. Просмотр каталога спортивных товаров.
2. Карточка товара с описанием, ценой, остатком и отзывами.
3. Добавление товаров в корзину и оформление заказа.
4. Личный кабинет покупателя с данными, заказами, избранным и просмотрами.
5. Кабинет менеджера для просмотра заказов и контроля остатков.
6. Админ-панель для управления товарами, категориями, заказами и сотрудниками.
7. Промокоды, отзывы и настройки сайта.
8. Проверка складских остатков при добавлении товара и оформлении заказа.

## Запуск

```bash
npm start
```

После запуска откройте сайт в браузере: [http://localhost:3030/](http://localhost:3030/).

## Проверка

```bash
npm test
```

Автоматическая проверка покрывает вход пользователей, каталог, фильтрацию,
создание заказа, уменьшение остатков, управление статусами, товарами,
категориями, настройками главной страницы и отзывами.
